# Cheikh Massamba Dia — Portfolio

Portfolio personnel de Cheikh Massamba Dia, développeur Full-Stack JavaScript et Chercheur.

Ce portfolio a été conçu pour présenter mon profil de développeur Full-Stack JavaScript, mes projets web, mon stack technique et mon parcours d'apprentissage. Il met également en valeur mon profil hybride, construit entre développement web, recherche de terrain, analyse des besoins réels et rédaction scientifique.

L'objectif du projet est de proposer une expérience fluide, responsive et professionnelle, avec un design dark navy premium, des effets glass, des animations soignées et une structure claire. L'interface accorde une attention particulière à l'accessibilité, à la lisibilité et au respect des préférences de mouvement de l'utilisateur.

## Projets présentés

### Kaolack Kitchen

Application web de commande en ligne conçue pour un service de restauration locale à Kaolack. Le projet permet aux clients de consulter un menu dynamique, d'ajouter des produits au panier, de passer une commande et d'en suivre le statut.

Fonctionnalités présentées :

- Menu organisé par catégories
- Panier responsive
- Passage et suivi de commande
- Dashboard d'administration
- Gestion des produits et catégories
- Upload d'images avec Cloudinary
- API REST avec validation et authentification

### Sama Horaire

Application web de réservation de tickets de transport. Elle permet de consulter les trajets disponibles, de réserver plusieurs places et de recevoir un ticket intégrant un QR code.

Fonctionnalités présentées :

- Consultation des trajets
- Réservation de places
- Ticket avec QR code
- Suivi de réservation
- Interface d'administration protégée
- Espace contrôleur et scan de QR code

Certains dépôts de projets sont privés et peuvent être présentés sur demande.

## Installation locale

Prérequis : disposer d'une version récente de Node.js et de npm.

```bash
git clone https://github.com/PapyDia/PortfolioPapy.git
cd PortfolioPapy
npm install
npm run dev
```

Après le lancement du serveur Vite, ouvrir l'adresse locale indiquée dans le terminal.

## Scripts disponibles

| Commande | Description |
| --- | --- |
| `npm run dev` | Lance le serveur de développement Vite |
| `npm run build` | Génère la version optimisée pour la production |
| `npm run lint` | Vérifie la qualité du code avec ESLint |
| `npm run preview` | Prévisualise localement la version de production |
